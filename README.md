## üîß Development Setup

### Prerequisites
- Ruby 2.7+
- Jekyll 4.3+
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/username/tretasharma.git
   cd tretasharma
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Start the development server:
   ```bash
   bundle exec jekyll serve
   ```

4. Open your browser to `http://localhost:4000`

### Building for Production

```bash
bundle exec jekyll build
```

The site will be generated in the `_site` directory.

## üöÄ GitHub Pages Deployment

This site is configured for automatic deployment to GitHub Pages:

1. **Repository Settings**: Enable GitHub Pages in repository settings
2. **Custom Domain**: Add `CNAME` file with domain name
3. **HTTPS**: Enable HTTPS enforcement in repository settings
4. **Automatic Deployment**: Pushes to main branch trigger automatic builds

### Custom Domain Setup

1. Add `CNAME` file to repository root:
   ```
   tretasharma.com
   ```

2. Configure DNS settings:
   - Add CNAME record pointing to `username.github.io`
   - Add A records for GitHub Pages IPs

3. Enable HTTPS in repository settings

## üìÑ Content Management

### Adding New Projects

1. Create new file in `_projects` directory
2. Add front matter with project details:
   ```yaml
   ---
   title: "Project Title"
   description: "Project description"
   status: "ongoing" # or "completed"
   image: "/assets/images/project-image.jpg"
   ---
   ```

### Updating Contact Information

Edit the `_config.yml` file to update contact details and social links.

### Adding New Experience

Update the timeline section in `index.html` with new experience entries.
